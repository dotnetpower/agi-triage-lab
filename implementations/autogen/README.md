# AutoGen 기반 구현

Microsoft AutoGen 프레임워크를 활용한 AGI Triage 구현체입니다.

## AutoGen 소개

[AutoGen](https://github.com/microsoft/autogen)은 다중 에이전트 대화를 지원하는 프레임워크입니다.

## 설치

```bash
pip install pyautogen
```
