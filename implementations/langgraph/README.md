# LangGraph 기반 구현

LangChain의 LangGraph를 활용한 AGI Triage 구현체입니다.

## LangGraph 소개

[LangGraph](https://github.com/langchain-ai/langgraph)는 상태 기반 멀티액터 애플리케이션을 구축하기 위한 라이브러리입니다.

## 설치

```bash
pip install langgraph
```
