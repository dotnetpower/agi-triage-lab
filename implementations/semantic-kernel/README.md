# Semantic Kernel 기반 구현

Microsoft Semantic Kernel을 활용한 AGI Triage 구현체입니다.

## Semantic Kernel 소개

[Semantic Kernel](https://github.com/microsoft/semantic-kernel)은 AI 오케스트레이션 SDK입니다.

## 설치

```bash
pip install semantic-kernel
```
